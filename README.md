# Landing page (Vite + React)

Projeto estático para publicação no **GitHub Pages**.

## Requisitos
- Node.js 18+ (ou 20+)

## Rodar localmente
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

O build sai em `dist/`.

## Publicação no GitHub Pages
Recomendado usar GitHub Actions para gerar o build e publicar automaticamente.
